# GRINDWAVE Jukebox

Static GitHub Pages version of the GRINDWAVE generative radio player.

## Offline use

1. Download the repository ZIP from GitHub.
2. Extract/unzip it first.
3. Open `index.html` in a browser.
4. Tap/click play; browsers require a user gesture before audio can start.

This copy is intended to be offline-safe. The Google font links were removed from `index.html`; the browser uses system fallback fonts instead.
